# Ecoleta

Aplicação para permitir que pessoas descubram pontos de coleta de resíduos cadastrados próximos a sua localização.

Sistema completamente desenvolvido durante a Next Level Week 1, evento promovido pela RocketSeat

# Tecnologias utilizadas e conceitos explorados

- TypeScript

## Server
- [NodeJS](https://nodejs.org/)
- API REST
- Conexão e gerenciamento de database relacional utilizando [Knex](http://knexjs.org/)
- Validação de dados por middleware, utilizando celebrate
- Upload de imagens

## Web
- [ReactJS](https://reactjs.org/)
- Conexão com APIs REST
- Upload de imagens
- Single Page Application
- Mapas e geolocalização
- React Rooks

## Mobile
- [React Native](https://reactnative.dev/)
- Conexão com APIs REST
- Mapas e geolocalização
- Expo
- React Rooks
